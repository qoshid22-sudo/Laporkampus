LaporKampus
Deskripsi
LaporKampus merupakan aplikasi berbasis Android yang dirancang untuk mempermudah civitas akademika dalam menyampaikan laporan, keluhan, maupun usulan terkait fasilitas dan layanan di lingkungan kampus. Aplikasi ini menjadi media pelaporan yang praktis, cepat, dan terdokumentasi sehingga setiap laporan dapat diproses secara lebih efektif.

Melalui LaporKampus, pengguna dapat mengirimkan laporan secara digital, melampirkan informasi pendukung, memantau status penanganan laporan, serta memperoleh informasi mengenai perkembangan penyelesaian laporan. Dengan adanya aplikasi ini, proses penyampaian aspirasi dan penanganan permasalahan di lingkungan kampus menjadi lebih transparan, efisien, dan mudah diakses kapan saja melalui perangkat Android.

Fitur Utama
Login dan Registrasi Pengguna.
Membuat laporan baru.
Mengunggah bukti pendukung (foto jika tersedia).
Melihat daftar laporan yang telah dikirim.
Melihat status perkembangan laporan.
Mengelola profil pengguna.
Antarmuka yang sederhana dan mudah digunakan.
Cara Menjalankan Aplikasi
Clone repository GitHub.
Buka project menggunakan Android Studio.
Tunggu proses Gradle Sync hingga selesai.
Hubungkan perangkat Android atau jalankan Emulator.
Klik tombol Run untuk menjalankan aplikasi.
